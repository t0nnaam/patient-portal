# patient-portal

A web portal letting patients book medical or dental appointments online
Admins manage schedules, appointment types, and availability
Built with React, Node.js, and PostgreSQL
(Project for INF 117)

Installation


If you have Node.js installed, you can spin up a server instantly from your terminal.

1. Open your terminal or command prompt.
2. Navigate to your folder: cd patient-portal
3. Run this command to start a temporary server: `npx http-server`
4. Open the http://localhost:8080 link provided in your terminal.
